# 🚀 Rock vs Mine Prediction using Machine Learning

## 📖 About the Project

This project uses Machine Learning to classify sonar signals as either **Rock (R)** or **Mine (M)**. It demonstrates a complete machine learning workflow, starting from loading and exploring the dataset to training a model, evaluating its performance, and making predictions on new data.

The project is built using **Python** and **Scikit-learn**, with **Logistic Regression** as the classification algorithm.

---

## 📊 Dataset

The project uses the **Sonar Dataset**, which contains sonar signal measurements reflected from different underwater objects.

- **Total Samples:** 208
- **Features:** 60 numerical attributes
- **Target Classes:**
  - **R** – Rock
  - **M** – Mine
